# Awesome Dotfiles
## Personal Dotfiles for AwesomeWM config
### Dependencies
- awesomewm 4 (or later)
- picom
- xfce4-goodies
- pulseaudio
- alsa
- acpi
- streetturtle volume widget

### Credits
- For the volume widget: https://github.com/streetturtle/awesome-wm-widgets/tree/master/volumebar-widget

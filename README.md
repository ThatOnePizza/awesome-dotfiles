### Dependencies
- awesomewm 4 (or later)
- picom
- pulseaudio
- alsa
- acpid
- streetturtle volume widget
- streetturtle brightness widget
- deficient battery widget
- Arc icons
- Alacritty
- zsh/ohmyzsh
- xscreensaver
- flameshot
- powerline-fonts
